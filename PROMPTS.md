# AI Prompt – Global Hackathon Challenge Section

You are a professional UI/UX designer and frontend developer.

Create a modern "Global Hackathon Challenge" section for a developer conference
website called DEVCONF 2026.

Requirements:

- Generate semantic HTML5 only.
- Include:
  - Section title
  - A short engaging description (2–3 sentences)
  - Prize Pool
  - Team Size
  - Duration
  - Registration Deadline
  - Difficulty Level
  - Winning Benefits
  - One Call-to-Action button

Design Requirements:

- The information should be displayed using six information cards.
- Arrange the cards in a responsive 3-column CSS Grid layout.
- Place the call-to-action button below the cards and center it.
- Use a clean, modern technology conference design with rounded corners,
  consistent spacing, and a blue accent color that matches the rest of the
  website.

CSS Requirements:

- Generate a separate CSS stylesheet.
- Use CSS Grid for the cards.
- Add a light background with subtle shadows.
- Add smooth hover effects for the cards and button.
- Use clean typography and proper spacing.
- Do not use any CSS frameworks.

Animation Requirements:

- Use only pure CSS animations.
- Do not use JavaScript.
- Add a smooth fade-in animation for the entire section when the page loads.
- Add a subtle lift and shadow effect when hovering over each information card.
- Add a gentle scale effect to the CTA button on hover.
- Keep all animations smooth, professional, and not distracting.

Content Guidelines:

- Use realistic and professional content.
- Keep the text concise.
- Do not use Lorem Ipsum.
- Make the content unique and suitable for a global developer conference.

Output Requirements:

1. Semantic HTML5
2. CSS only
3. No JavaScript
